danardos.com

A Pizza Restaurant Website & Point of Sale System

Danardos.com is website built for the new owners of D'anardo's Pizza. It includes landing pages, a fully featured ordering system, and a POS system for employees to place orders and view/change the status of previous orders. Danardos.com is an asp.net core project built from the ground up using vue.js. I'm happy with the result and the system is now live.

ASP.NET Core | Vue.js | Azure | SQL
